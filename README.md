# RecipeHub

RecipeHub is a modern full-stack recipe sharing platform where users can discover, create, save, 
and review recipes.

This project is being developed as a personal portfolio project to demonstrate my full-stack 
software engineering skills using modern web technologies.

**Project Status:** This project is currently under active development. New features 
and improvements are continuously being added.

---

## Features

### Authentication
- User Registration
- User Login
- Secure Authentication with Laravel Sanctum

### Recipes
- Create Recipes
- View Recipes
- Edit Recipes
- Delete Recipes
- Recipe Categories
- Recipe Tags

### Search & Filtering
- Search recipes by title
- Filter by category
- Filter by tags

### Community Features
- Favorite Recipes *(In Progress)*
- Ratings & Reviews *(In Progress)*
- User Profiles *(Planned)*
